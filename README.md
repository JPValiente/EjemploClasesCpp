# EjemploClasesCpp
Ejemplo de clases en C++

<h2>Como compilar?</h2>

El repositorio cuenta con 3 archivos, un archivo main y dos archivos de la clase que se dividen en un cpp y su header. 

Para compilar se necesita: 

g++ EjemploC++.cpp Persona.cpp Persona.h

y ejecutarlo con:

./a.out
